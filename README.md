# NICAT

NICAT isn't a computer assisted translator

# Situation

有时候想做一些简短的翻译，懒得用正式的 CAT 软件开档案，而且想快速生成对比图。

搜了一下好像没看到这样的小工具，遂开始重复造轮。

# Not feature

- 机器辅助翻译，因为这不是一个机器辅助翻译应用
